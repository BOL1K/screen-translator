# Проект: экранный переводчик-помощник для изучения языков

## Что строим (цель мечты)
Приложение под Windows: пользователь наводит на слово, жмёт горячую клавишу, 
получает контекстный ИИ-перевод, и слово сохраняется в личный словарь 
(вместе с предложением и скриншотом) для повторения.

## MVP (строим ПЕРВЫМ, не забегая вперёд)
1. Пользователь вводит предложение и слово из него.
2. Программа отправляет их в Gemini API и получает перевод СЛОВА с учётом контекста.
3. Слово + перевод + предложение сохраняются в локальный словарь.
4. Пользователь может открыть словарь и посмотреть сохранённые слова.

## Стек
- Язык: C# / .NET
- UI: WinUI 3
- Перевод: Google Gemini API (бесплатный тариф)
- Хранение словаря: SQLite (или простой JSON-файл на старте)
