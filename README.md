# Экранный переводчик-помощник

Программа для Windows, которая помогает учить английский язык прямо во время чтения, игр или визуальных новелл. Наводишь на слово (или курсор, или прямо в игре без курсора) — получаешь перевод с учётом контекста, и слово можно сохранить в свой личный словарь для повторения.

Перевод делает бесплатный ИИ Google Gemini.

## Что нужно для запуска

1. **.NET** (актуальная версия) — [скачать здесь](https://dotnet.microsoft.com/download).
2. **Бесплатный API-ключ Gemini** — получить его можно в [Google AI Studio](https://aistudio.google.com/).
3. Задать ключ в переменную окружения `GEMINI_API_KEY`. В PowerShell это делается так:

   ```
   setx GEMINI_API_KEY "твой_ключ_сюда"
   ```

   После этого нужно закрыть и заново открыть терминал, чтобы переменная подхватилась.

4. Запуск программы:

   ```
   dotnet run
   ```

Ключ **никогда** не хранится в коде — программа берёт его только из этой переменной окружения.

## Горячие клавиши (работают в любой момент, даже если программа свёрнута)

- **Alt+Q** — перевести слово под курсором мыши и показать перевод.
- **Alt+E** — перевести слово под курсором и сразу сохранить его в словарь.
- **Alt+W** — открыть прозрачный оверлей с рамками вокруг слов на экране (удобно для игр и визуальных новелл, где не видно обычного курсора). Внутри оверлея:
  - левый клик по слову — показать перевод;
  - правый клик по слову — перевести и сохранить в словарь;
  - **Esc** — закрыть оверлей.

## Меню в консоли

- **1** — новый перевод (вручную ввести предложение и слово).
- **2** — посмотреть словарь сохранённых слов.
- **3** — выход из программы.
- **4** — выбрать модель перевода и посмотреть статистику запросов за сеанс.

## Где что хранится

- `dictionary.json` — твой личный словарь (слово, перевод, часть речи, пояснение, предложение и его перевод). Создаётся автоматически при первом сохранении.
- `screenshots/` — скриншоты, сделанные горячими клавишами. Создаётся автоматически.

Оба этих места — только на твоём компьютере, в git не попадают.
